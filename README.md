# 大航海時代3 スフィンクスクイズ攻略ツール

大航海時代3のスフィンクスクイズを解くための計算ツールです。

## 機能

- 第1問：有名なスフィンクスの謎かけ（朝4本脚、昼2本脚、晩3本脚）の答えを表示
- 第2問：4本脚と2本脚の怪物の匹数を計算（総匹数と総脚数から）
- 第3問：成長を伴う複雑な計算問題（4本脚が2本脚に成長、一部は3本脚に）
- 第4問：新規誕生を含む最も複雑な計算問題

## インストール

```bash
pnpm install
```

## 使用方法

```bash
pnpm dev
```

ブラウザで <http://localhost:3000/> にアクセスしてください。

## 計算式

### 第2問

- 4本脚: (B ÷ 2) - A
- 2本脚: 2A - (B ÷ 2)

### 第3問

- 4本脚であった者: (A + B - C) ÷ 2

### 第4問

- 4本脚であった者: (C - A - B) ÷ 6

## ビルド
